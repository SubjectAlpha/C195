Customer Appointment Application v1 02/25/2023
The purpose of this application is to allow users to log in and manage appointments between customers and their contacts.
Contact: Jacob Starr, jstar56@wgu.edu
IntelliJ IDEA Community Edition 2021.1.3 x64
Java SE 17.0.1
OpenJFX-11.0.1
mysql-connector-java-8.0.25
Report 3 description: My third report is to display the amount of new customers that month.
Directions:
Import the project file in to your IntelliJ
Set the 'Main' file in src.exam to be the launch source
Import both the JavaFX library and the MySQL connector library to your project
Ensure your Java version is set to 17.0.1 in your launch arguments
Your project should now be ready to run.

Running MySQL reports:
Reports 1 and 3 can be run as normal SQL queries.
To execute report 2 you must first add the stored procedure to your server
To generate report 2 you can then use the code under the 2nd comment to view the results of the stored procedure
